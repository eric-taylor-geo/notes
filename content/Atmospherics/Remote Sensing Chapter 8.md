# 8.1 Cloud Remote Sensing

- Clouds are often a hindrance for surface remote sensing. However, they are also a prime indicator for the conditions of the atmosphere.

## Cloud Top Temperature

- Accurate cloud top temperature and height formation is important for aviation forecasting and for early warning thunderstorm development
	- When cloud top temperatures get colder, it means that the cloud is getting higher in the atmosphere, reflecting the uplift of warm, moist air.
	- Vice versa for weakening
- For optically thick clouds, weather satellite imagery provides a reasonable estimate of cloud top temperature - atmospheric absorption above the cloud can be neglected and the brightness temperature of a thermal, window channel can be taken as the thermodynamic cloud top temperature
- Optically transparent clouds differ - directly using the measured thermal brightness will overestimate the true top cloud temperature resulting in an underestimation of cloud height
- CTTs are first retrieved assuming that clouds are black bodies 

**Orbits**

- Geostationary satellites can be used to compute cloud top temperatures from their infrared channels
